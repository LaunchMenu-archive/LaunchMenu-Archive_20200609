ChannelSender

new ChannelSender(ID, subChannelID, senderID)

Create a new channel sender, allowing to send messages to the channel
Parameters
ID:string
The unique identifier for the channel
subChannelID:string
The subChannelID that will be used to access special subchannel methods
senderID:string
An ID that the reciever of this channel can respond to (can be left out)
